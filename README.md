# Investments Platform

A lightweight investment analysis platform built with **Next.js** that provides real-time equity dashboards, risk analysis, and valuation tools. Built as a personal dashboard for fast, repeatable decision-making with clean data handling and extensibility in mind.

---

## Features

- **Equity Dashboards**
  - Interactive price charts
  - Short- and long-horizon returns (1D, 1W, 1M, 1Y)
  - Annualized volatility and performance metrics

- **Monte Carlo Risk Modeling**
  - Probabilistic price simulations based on historical returns
  - Distribution-based downside and upside analysis (Interactive Histogram)

- **Market Data API**
  - Server-side data fetching via custom API routes
  - Secure handling of third-party market data APIs
  - No client-side exposure of API keys
